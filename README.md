## About the Project
This project was completed as part of the Digital Literacy course. The main objective of this project is to understand and apply essential digital skills required in today’s modern world. It focuses on building a professional online presence, using digital tools effectively, communicating professionally, and staying safe online.

---

## Tasks Completed

Task 1: Digital Literacy Infographic  
Created a one-page infographic using Canva explaining digital literacy, its importance, safe internet practices, and email etiquette.

Task 2: Digital Portfolio  
Developed professional profiles on platforms like GitHub, LinkedIn, and Kaggle to establish a strong digital presence.

Task 3: Coding & Collaboration Platforms  
Solved a beginner-level Python problem on HackerRank and created a Google Form quiz on digital literacy awareness.

Task 4: Email Etiquette & Social Media Guide  
Wrote two professional emails and created a checklist of Do’s and Don’ts for responsible social media usage.

Task 5: Cybercrime Awareness  
Prepared a case study on cybercrime (UPI fraud) and created a prevention checklist to promote safe online practices.
